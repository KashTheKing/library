# Bounding Box
Make a part that matches a model's bounding box, in one click.

I made this plugin because I find myself needing to make hitboxes for models very often. This cuts the process down into just the press of a button.

## Installation
Get the plugin
https://create.roblox.com/store/asset/83615267790769/Bounding-Box-Plugin

## How to use:
1. Select a model.
2. Click the "Bounding Box" toolbar button.

## Optional configuration:
If you'd like to change the default properties of new bounding boxes:
1. Select a bounding box (or any part)
2. Click the "Match Properties" toolbar button.

Future bounding boxes will now have the same properties (i.e. Anchored, CanCollide, CanTouch, etc.) as the part you selected.
