# Studio Wally
Search and download Wally packages, without ever leaving Roblox Studio

I made this plugin because I wanted to continue fewkz' deprecated Studio Wally plugin and add package types.
My fork of his plugin re-exports types from installed packages, and adds some general bug fixes.

## Installation
Get the plugin
https://create.roblox.com/store/asset/133519889832441/Studio-Wally-With-package-types

## How to use:
1. Open the widget
2. Search a wally package (i.e. "kashtheking/binder")
3. Click the download button and choose a download option. The "Shared module" is recommended for almost all packages.
