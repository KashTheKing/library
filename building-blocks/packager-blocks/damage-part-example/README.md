# Damage Part Example
A simple damage part that uses [Binder](https://github.com/KashTheKing/library/tree/main/packages/src/Binder)

## Installation:
> Note: This building block uses [Packager](https://github.com/KashTheKing/library/tree/main/plugins/packager) for installation

- Download the [rbxm](https://github.com/KashTheKing/library/tree/main/building-blocks/packager-blocks/damage-part-example/damage-part-example.rbxm)
- Insert the rbxm into Roblox Studio
- Use [Packager](https://github.com/KashTheKing/library/tree/main/plugins/packager) to unpack it

## Notes:
The module can be placed anywhere, just make sure to require it!
